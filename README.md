# Typescript API

Api completa restful scritta interamente in typescript in node utilizzando express come web server.


## Run Locally

Clone the project

```bash
  git clone https://github.com/gherardi/crud-ts
```

Go to the project directory

```bash
  cd crud-ts
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
